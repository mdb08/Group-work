Monopoly Bored Game

You will each be given a slightly different version of this game, for which you should design
and code an Object-Oriented solution using Java.

The task as written is broken up into 5 phases. You will be assessed at these points:

End of Phase 1

End of Phase 3

End of Phase 5

Week 5

Week 8

End of Week 11

The first two assessments will be in the form of in-class assessments performed within the
tutorial sessions. The third assessment will be by presentation of your final program and
documentation.
